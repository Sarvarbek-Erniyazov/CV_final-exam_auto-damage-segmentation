# Auto Damage Segmentation

## Loyiha haqida
Mashina qismlarini segmentatsiya qilib, shikastlangan joylarni aniqlash tizimi.

## Texnologiyalar
- YOLOv8-seg
- FastAPI
- Roboflow
- ONNX

## Struktura
- dataset/ - train/val/test rasmlari
- model/ - best.pt va best.onnx
- api/ - FastAPI kodi
- notebook/ - Colab notebook
- report/ - Hisobot
